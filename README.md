# setup configuration for linux system.


## How to run.
```console
foo:~$ bash ./linux-setup.sh
```


- Vim Configuration

| Key Configuration  | Operation				   |
| ------------------ | --------------------------- |
| Ctrl + X			 | Save & Quit				   |
| Ctrl + Q			 | Force Quit				   |
| Ctrl + S			 | Save						   |
| Ctrl + N			 | New Tab					   |
| F5				 | Open Terminal			   |
| F4 in latex		 | Start Automatic Compilation |
| F5 in latex		 | Compile File				   |
